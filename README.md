# Mr.Rm19 Shield 

Alat sederhana untuk memproteksi kode file web (HTML, JavaScript, dan CSS) agar tidak mudah disalin atau dimodifikasi oleh orang lain.

## Fitur Utama
* **Enkripsi Kode**: Mengamankan logika file web menggunakan teknik fragmentasi.
* **Security Guard**: Proteksi klik kanan dan pembersihan konsol otomatis.
* **Akses Terbatas**: Memerlukan password administratif untuk menjalankan alat.

## Cara Penggunaan
1. Jalankan aplikasi `Mr.Rm19_Shield.exe`.
2. Masukkan password akses sistem. hanya Mr.Rm19 yang tahu atau chat email ku dan kontak ku di facebook 
3. Masukkan kunci enkripsi untuk file yang ingin diproteksi.
4. Pilih file sumber dan simpan hasilnya.
